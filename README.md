# UIme – User Interface Module (v1.0)

UIme ist ein RAW‑UI‑Operator ohne eigene Logik.
Es dient als Einstiegspunkt in das Raster-System.

## Eigenschaften
- Inhalt: leer
- Achsen: keine
- Marker: keine
- GEN/EICH/ID: keine
- MODE: RAW
- Rolle: UI‑Slave
- Funktion: reine Darstellung

## Zweck
UIme stellt UI‑Einstiegspunkte bereit:
- male
- modi
- UIme
- root

UIme führt keine Berechnungen aus und besitzt keine Engine.

## Struktur
UIme besteht aus einer index.html, die Raster-Bewegung darstellt.

## Kompatibilität
- NC-kompatibel
- RAW-neutral
- SYNC-fähig (Mitlauf)
- RESPO-kompatibel
- tri5mix-kompatibel
- ERROR-kompatibel

## Status
UIme ist ein 0‑MANN‑UI‑Modul.
Es bleibt leer und neutral.
